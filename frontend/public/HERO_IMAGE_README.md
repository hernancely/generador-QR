# Imagen del Hero

Para personalizar la imagen del Hero (banner principal), coloca tu foto de boda en:

```
frontend/public/hero-wedding.jpg
```

## Especificaciones Recomendadas

- **Formato**: JPG o PNG
- **Dimensiones recomendadas**: 1920x1080 píxeles (proporción 16:9)
- **Tamaño de archivo**: Máximo 2MB para mejor rendimiento
- **Orientación**: Horizontal (landscape)

## Consejos

- Usa una imagen con buena iluminación
- Asegúrate de que las caras estén centradas o en el tercio medio
- La imagen tendrá un overlay oscuro para mejor legibilidad del texto
- Evita imágenes muy oscuras o con mucho detalle que distraiga

## Alternativa

Si no tienes una imagen lista, puedes usar cualquier foto temporal con el nombre `hero-wedding.jpg` en la carpeta `public`.
